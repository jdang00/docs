## Built for focused studying

LearnTerms is a cohort-based study platform designed to help you spend less time organizing and more time learning. The product keeps classes, modules, questions, and progress tightly connected so every study session stays relevant to the material you actually need to know.

The docs follow that same principle. Start with onboarding and local setup, then move into the workflows you use most: studying modules, managing content, and generating questions from source material.

## What belongs here

These docs should explain the real LearnTerms product as it exists in the app:

- Student workflows like classes, modules, tests, and progress
- Admin workflows like content ingestion, question curation, and cohort management
- AI-assisted workflows like chunking and LearnTerms generation models
- Local development and docs contribution guidance for the Mintlify site
