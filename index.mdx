## Start with the real product

LearnTerms is a cohort-based study platform. It organizes material by cohort, semester, class, and module so your study time stays tied to the questions your group actually needs.

These docs follow that structure. Start with the onboarding and architecture pages if you need orientation, then move into the student, admin, or contributor workflow that matches what you are doing.

## What these docs cover

The goal of this site is to document the shipped LearnTerms product as it exists today:

- Student workflows like classes, modules, saved progress, and custom tests
- Admin workflows like cohort organization, content ingestion, and AI-assisted generation
- Contributor workflows like writing, editing, and attaching question media
- Platform details like roles, auth, uploads, and system dependencies
