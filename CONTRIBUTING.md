# Contributing to LearnTerms docs

## Local workflow

1. Clone this repository.
2. Run `npx mint dev` from the repo root.
3. Open `http://localhost:3000`.
4. Edit the relevant `.mdx` page or `docs.json`.
5. Run `npx mint broken-links` before you push.

## Content workflow

1. Confirm the product behavior in `/Users/justindang/learnterms`.
2. Port or rewrite the guidance in MDX for the Mintlify site.
3. Keep navigation aligned to real LearnTerms workflows, not template categories.

## Writing guidelines

- Use active voice.
- Address the reader as `you`.
- Prefer real product terms over generic documentation language.
- Keep instructions sequential and easy to scan.
- Include examples when a workflow would otherwise feel abstract.
